🐶🐱 Galería de Perros y Gatos

Bienvenido a la Galería de Perros y Gatos, una aplicación que te permite explorar imágenes de estas adorables mascotas en distintas razas. Puedes ver imágenes aleatorias o buscar por raza específica.

Utilizamos las API de Dog CEO y The Cat API para obtener imágenes de alta calidad.
